# Umbrella - Build Guide

Руководство по сборке Umbrella в исполняемый файл (.exe)

## Быстрая сборка (Рекомендуется)

Запустите:
```bash
.\build_umbrella.bat
```

Это создаст **папку** `dist/Umbrella/` с:
- Umbrella.exe (главный файл)
- Все необходимые DLL и зависимости
- Иконка Umbrella Corporation

## Альтернатива: Один файл

Если хотите один EXE файл (может не работать):
```bash
.\build_onefile.bat
```

## Что делает build_umbrella.bat

1. Установит PyInstaller и Pillow
2. Создаст иконку Umbrella Corporation (красный зонтик)
3. Соберет приложение в папку
4. Поместит результат в `dist/Umbrella/`

## Результат

После успешной сборки:
- **Папка:** `dist/Umbrella/`
- **Главный файл:** `dist/Umbrella/Umbrella.exe`
- **Размер папки:** ~100-150 MB
- **Иконка:** Красный логотип Umbrella Corporation
- **Запуск:** Двойной клик на Umbrella.exe

## Распространение

Для распространения:
1. Скопируйте всю папку `dist/Umbrella/`
2. Отправьте пользователям
3. Они запускают `Umbrella.exe` из этой папки

**ВАЖНО:** Нужна вся папка, не только .exe файл!

## Структура сборки

```
Cwelium-main/
├── Umbrella.spec          # Конфигурация PyInstaller
├── build_umbrella.bat     # Скрипт автоматической сборки
├── create_icon.py         # Генератор иконки
├── umbrella.ico           # Иконка (создается автоматически)
├── build/                 # Временные файлы сборки
└── dist/
    └── Umbrella.exe       # Готовый исполняемый файл
```

## Особенности

- **Без консоли:** Запускается как GUI приложение
- **Одним файлом:** Все зависимости упакованы внутри
- **Портативный:** Можно запускать на любом Windows без Python
- **UPX сжатие:** Уменьшенный размер файла

## Требования

- Windows 10/11
- Python 3.8+
- ~200 MB свободного места для сборки

## Распространение

После сборки можно распространять только `Umbrella.exe`:
- Не требует установки Python
- Не требует установки зависимостей
- Работает "из коробки"

## Troubleshooting

**Ошибка: "PyInstaller not found"**
```bash
..\venv\Scripts\python.exe -m pip install pyinstaller
```

**Ошибка: "PIL not found"**
```bash
..\venv\Scripts\python.exe -m pip install pillow
```

**Большой размер файла:**
- Это нормально, EXE включает весь Python runtime
- Можно уменьшить, исключив неиспользуемые модули в .spec файле
