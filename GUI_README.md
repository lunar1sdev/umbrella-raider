# Umbrella GUI Edition

Графический интерфейс для Umbrella Raider с полным функционалом.

## Запуск

1. Установите зависимости (если еще не установлены):
```bash
.\install.bat
```

2. Запустите GUI:
```bash
.\run_gui.bat
```

Или напрямую:
```bash
..\venv\Scripts\python.exe Cwelium_GUI.py
```

## Возможности

### Token Management (Управление токенами)
- Загрузка токенов из файла
- Проверка валидности токенов
- Форматирование токенов (email:pass:token → token)
- Добавление токенов вручную

### Server Actions (Действия с серверами)
- **Joiner** - Присоединение к серверам
- **Leaver** - Выход с серверов
- **Message Spammer** - Спам сообщениями
  - Поддержка массового пинга (Mass Ping)
  - Случайные строки для обхода фильтров
  - Настраиваемая задержка
- **Thread Spammer** - Создание спам-тредов

### User Actions (Действия с пользователями)
- **DM Spam** - Спам в личные сообщения
- **Call Spam** - Спам звонками
- **Bio Changer** - Изменение био профиля
- **Nickname Changer** - Массовая смена никнеймов
- **Friend Spam** - Спам заявками в друзья
- **Typing Effect** - Эффект печатания

### Voice Actions (Голосовые действия)
- **Voice Joiner** - Присоединение к голосовым каналам
- **Onliner** - Установка статуса "онлайн"
- **Soundboard Spam** - Спам звуками из саундборда

### Advanced (Расширенные)
- **Accept Rules** - Принятие правил сервера
- **Onboarding Bypass** - Обход онбординга
- **Guild Checker** - Проверка доступа к серверу
- **Reactor** - Добавление реакций на сообщения
- **Button Click** - Нажатие кнопок на сообщениях

## Новые возможности

### Кнопка Info
- Полное руководство по всем функциям
- Инструкции по получению ID
- Советы по использованию
- Нажмите кнопку "Info" в верхней части окна

### Смена языка
- Поддержка английского (EN) и русского (RU)
- Переключатель в верхней части окна
- Автоматическое сохранение выбора

## Интерфейс

- **5 вкладок** - разные категории функций
- **Лог активности** - отображение в реальном времени
- **Статистика** - количество токенов и прокси
- **Прокручиваемые вкладки** - для удобства навигации

## Как получить ID

1. Включите режим разработчика:
   - Настройки → Расширенные → Режим разработчика (ВКЛ)

2. Получение ID:
   - **Server ID**: ПКМ на иконку сервера → Копировать ID
   - **Channel ID**: ПКМ на канал → Копировать ID
   - **User ID**: ПКМ на пользователя → Копировать ID
   - **Message ID**: ПКМ на сообщение → Копировать ID

## Советы

- Используйте прокси для избежания лимитов
- Регулярно проверяйте токены
- Используйте задержки в спамере
- Mass Ping требует скрапинга участников (занимает время)
- Некоторые функции требуют наличия токенов на сервере

## Примечания

- Все действия выполняются в отдельных потоках
- Логи отображаются в реальном времени
- Токены и прокси загружаются из `data/tokens.txt` и `data/proxies.txt`
- Настройки сохраняются в `config.json`
